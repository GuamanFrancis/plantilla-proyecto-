ENDPOINTS DEL BACKEND
─────────────────────
  POST   /api/auth/login          → { email, clave }  →  { token, user }
  GET    /api/auth/me             → (necesita token)  →  { user }
  POST   /api/auth/logout         → (necesita token)  →  { message }

  GET    /api/pacientes           → lista de pacientes (con sus citas)
  POST   /api/pacientes           → { nombre, apellido, cedula, telefono, email? }
  PUT    /api/pacientes/:id       → campos opcionales
  DELETE /api/pacientes/:id       → elimina

  GET    /api/especialidades      → lista de especialidades
  POST   /api/especialidades      → { nombre, descripcion? }
  PUT    /api/especialidades/:id  → campos opcionales
  DELETE /api/especialidades/:id  → elimina

  GET    /api/citas               → lista de citas (con paciente y especialidad incluidos)
  POST   /api/citas               → { pacienteId, especialidadId, fecha, motivo? }
  DELETE /api/citas/:id           → elimina  (⚠️ Citas NO tiene PUT/update)

FORMA DE LAS RESPUESTAS DEL BACKEND
────────────────────────────────────
  Éxito:  { success: true,  data: <objeto o array> }
  Error:  { success: false, error: { code, message, details? } }
  OJO:    Para extraer la data → res.data.data  (dos .data: uno Axios, uno envelope)

PATRONES IMPORTANTES (para recordar en el examen)
──────────────────────────────────────────────────
1. handleChange GENÉRICO (un solo handler para todos los inputs):
   function handleChange(e: React.ChangeEvent<HTMLInputElement>) {
     const { name, value } = e.target
     setFormData(prev => ({ ...prev, [name]: value }))
   }
   ⚠️  El input DEBE tener el atributo name="<campo>" igual al nombre en formData

2. Crear VS editar con un solo handleSubmit:
   if (editingId !== null) {
     await updatePaciente(editingId, formData)  // editar
   } else {
     await createPaciente(formData)             // crear
   }

3. Doble .data al retornar de api:
   const res = await client.get('/pacientes')
   return res.data.data   // res.data = envelope, .data = el array/objeto real

4. useEffect para cargar lista al montar la página:
   useEffect(() => { loadItems() }, [])    // [] = solo una vez al montar

5. Selects en CitasPage cargan datos reales:
   const [pacientes, setPacientes] = useState<Paciente[]>([])
   const [especialidades, setEspecialidades] = useState<Especialidad[]>([])
   useEffect(() => { Promise.all([getPacientes(), getEspecialidades()]).then... }, [])
