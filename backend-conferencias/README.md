# Backend - Sistema de Gestión de Conferencias (Caso 5)

## Entidades
- **Conferencista**: nombre, apellido, email?, telefono?, especialidad?
- **Auditorio**: nombre, capacidad, ubicacion?
- **Reserva**: conferencistaId, auditorioId, fecha, tema? (relacional)

## Cómo correr
```bash
npm install
npm run setup
npm run dev
```
