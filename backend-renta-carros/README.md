# Backend - Sistema de Renta de Carros (Caso 2)

## Entidades
- **Cliente**: nombre, apellido, cedula, telefono, email?
- **Vehiculo**: marca, modelo, placa, anio?, descripcion?
- **Reserva**: clienteId, vehiculoId, fechaInicio, fechaFin, descripcion? (relacional)

## Cómo correr
```bash
npm install
npm run setup
npm run dev
```
