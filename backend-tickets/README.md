# Backend - Sistema de Tickets de Asistencia Técnica (Caso 4)

## Entidades
- **Tecnico**: nombre, apellido, especialidad, email?, telefono?
- **Cliente**: nombre, apellido, cedula, telefono, email?
- **Ticket**: clienteId, tecnicoId, descripcion, estado, prioridad (relacional)

## Cómo correr
```bash
npm install
npm run setup
npm run dev
```
